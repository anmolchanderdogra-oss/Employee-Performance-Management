Implementation Code

// Get session
HttpSession session = request.getSession(true);

// 1. Get creation time
long creationTime = session.getCreationTime();

// 2. Get last accessed time
long lastAccessedTime = session.getLastAccessedTime();

// 3. Set timeout (30 minutes)
session.setMaxInactiveInterval(1800);

// 4. Store user data
session.setAttribute("userId", userId);
session.setAttribute("username", username);

// 5. On logout - invalidate
session.invalidate();

Concepts Demonstrated

Static vs Dynamic Web
  • This page is dynamic - content changes based on session
  • Session times update with each request
  • Static pages show same content to all users
Client-Server Architecture
  • Client: Browser sends HTTP request
  • Server: Servlet processes request
  • Session: Server-side storage
Session vs Persistent Storage
  • Session: Temporary, user-specific
  • Database: Permanent records
  • Session expires, data persists
